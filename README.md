## 📋 完整测试流程

### 第1步：输入生日

1. 选择你的出生年份（1920-2026）
2. 选择月份（1-12月）
3. 选择日期（1-31日）
4. 点击"开始探索"按钮

### 第2步：查看先天五行

系统会显示：
- 你的日干（甲/乙/丙/丁/戊/己/庚/辛/壬/癸）
- 你的五行属性（木🌳/火🔥/土⛰️/金⚙️/水💧）
- 五行属性描述

### 第3步：完成性格测试

5道精心设计的题目，每题5个选项：
- 自然景观偏好
- 性格描述
- 颜色偏好
- 团队角色
- 压力应对

点击选项会自动进入下一题

### 第4步：查看推荐城市

系统会显示：
- 推荐的城市名称（如：杭州、上海、北京等）
- 你的五行属性标签
- 匹配度百分比（85-98%）
- 城市详细解读

## 🐛 常见问题

### Q: 为什么推荐的城市每次都不一样？

A: 因为每个五行有3个对应城市，系统会随机选择一个。这是为了增加趣味性和多样性。

### Q: 算法准确吗？

A: 天干地支算法已经过验证，与在线万年历一致。五行相生相克理论也是传统的。

### Q: 可以保存结果吗？

A: MVP版本没有保存功能。完整版将支持历史记录保存。

### Q: 为什么只有5道题？

A: MVP是为了快速展示完整流程。完整版将有49道题目，提供更准确的分析。
