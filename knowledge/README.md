## 🎯 如何使用本知识库

### 场景1：开始新项目

**步骤**：
1. 阅读"项目经验知识库.md"的"开发流程"部分
2. 根据"技术栈选择"章节确定技术方案
3. 参考"开发经验"进行项目规划
4. 使用"快速参考指南.md"的命令快速初始化

### 场景2：部署新项目

**步骤**：
1. 阅读"快速参考指南.md"的"5分钟部署流程"
2. 复制vercel.json配置模板
3. 按照部署步骤操作
4. 遇到问题时查看"项目经验知识库.md"的"常见问题解决"部分

### 场景3：遇到问题

**步骤**：
1. 查看"快速参考指南.md"的"快速故障排查"
2. 在"项目经验知识库.md"中搜索相关问题
3. 参考"解决方案"部分
4. 如果仍无法解决，查阅官方文档

### 场景4：优化现有项目

**步骤**：
1. 查看"快速参考指南.md"的"性能优化清单"
2. 参考"移动端优化清单"
3. 使用"常用代码片段"
4. 部署前使用"部署前检查清单"
